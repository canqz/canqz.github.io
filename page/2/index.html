思路

  2. 查找：循环判断是否溢出或者找到，最后用if判断，返回值为int型，是location
  3. 插入: 先判断数组是否满了，再判断插入的位置是否是正确的位置，然后采用后面先替换原则，腾出位置插入，表厂++
  4. 删除: 先判断插入位置是否正确，然后采用前面先替换原则，表长- -

基础概念

  1. JDK(Java Development Kit)
    JRE(Java Runtime Environment)
    JVM(Java Virtual Machine)
  2. 程序写好后JDK先编译.java文件形成字节码.class文件，然后JRE JVM解释.class文件
  3. 类和对象：

    类：猫
    对象：这只猫

  4. ‘A’是字符（单引号） “A”是字符串（双引号）
  5. Java全部采用unicode编码机制，其中有汉字
  6. 小数默认为double型，float型后面要加上F（如3.14F）
  7. 类定义在包中
  8. 包裹类型
  9. 成员变量（会自动赋初始值0）与成员函数
  10. this就是这次调用函数（方法）的那个对象
  11. 构造函数:在构造的时候自动调用，先定义初始话的东西，再执行构造函数里面的东西，构造函数的名字和类的名字完全相同，构造函数无返回类型

Android安装模拟器Genymotion

下载Genymotion模拟器遇到的问题

为什么要下载Genymotion

  1. 小编Android studio自带的模拟器启动太慢
  2. 由于小编自己傻瓜式的在系统变量里”添加”了path变量，导致把以前的path变量全部覆盖了，自带的模拟器在最近的使用过程中出现了很多问题
  3. 自己也想体验一下Genymotion到底哪里好用

步骤

  1. 登入官网注册账号https://www.genymotion.com/
  2. 注册完后下载，注意一点:因为Genymotion运行需要VirtualBox，所以需要安装这个内存更大的版本
  3. 接下来傻瓜是安装
  4. 安装完成之后，会需要选择手机型号
  5. 我选择了三星s8 ，下载速度超级慢，还不断的下载失败，于是我去找了找网上的资料
  6. 大致过程如下
    1. 先去自己的c盘目录里找到C:\Users\Administrator\AppData\Local\Genymobile我找的时候真的没有看见那个AppData文件，后来还是通过搜索框找到的，真的没有看见，我找了好久
    2. 查看日志文件genymotion.log里面有这种字样:2019-10-19T23:24:05+08:00 [Genymotion:10928] [debug] GET “https://cloud.genymotion.com/vmtemplate/getall?genymotion_version=3.0.3" 复制其中的网址，去Google里面打开，然后他会自动下载，但是下载速度也是龟速，几k每秒。
    3. 对镜像了解不是很多，本来想通过镜像下载，但是镜像里根本就没有这个文件啊，后来用手机热点搭了梯子出去，用了好多流量呜呜呜
    4. 把下载好的文件复制到C:\Users\用户名\AppData\Local\Genymobile\Genymotion\ova里面。
    5. 接下来打开Genymotion，解压它就好了
  7. 这样之后在Android studio里面就可以看见自己刚刚安装的模拟器了，可以运行了
  8. 对于NASDK2.0_Studio的项目，我还是运行不了，版本号已经改正，编译也照常过了，就是不能运行到模拟器上，自己连了手机也不能运行。目前还没有找到解决办法
