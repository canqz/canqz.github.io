    函数的复写（override)

    1. 是在继承里面的，是对继承的修改
    2. 父子各有一个成员函数完全相同（返回值，函数名，参数列表都一样）
    3. super.introduce();这是调用父类的方法，super指的是父类；
      具体代码如下：里面的的eat函数就是函数的复写

    对象转型

    1. 向上转型：把子类变成父类（一定会成功）

      注意点1：一个引用能调用那些成员（变量，函数）取决于引用类型
      注意点2：一个引用调用的具体是哪种方法，取决于这个引用所指向的对象

    2. 向下转型：把父类变成子类

    抽象类与抽象函数

    1. 抽象函数：只有函数定义，没有函数体的函数。

    2. 使用abstract定义的类是抽象类，不能生成对象

    1. 抽象类就是用来继承的，当父亲，子类里面复写抽象函数。
    2. 如果一个类里面有抽象函数，那么这个类必须申明为抽象类
    3. 抽象类里面可以有构造函数；在生成子类的时候调用
    4. 作用之一：提示报错。
